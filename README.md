# Gitingest to JSON Converter

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Active-blue?logo=github&logoColor=white)](https://sincekmori.github.io/gitingest-to-json/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A lightweight, browser-based tool designed to transform [Gitingest](https://gitingest.com/) "Files Content" output into a clean, structured, and sorted JSON format.

**Live URL:** https://sincekmori.github.io/gitingest-to-json/

1. Go to [Gitingest](https://gitingest.com/) and ingest a repository.
2. Scroll to the **Files Content** section and copy the text.
3. Paste the text into this tool's input area.
4. Click **GENERATE JSON**.
5. Copy the resulting JSON for your LLM or application.
